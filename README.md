# KeybCompare v2

Next.js + Tailwind + shadcn-style UI version of the keyboard comparison site.

## Run locally

```bash
npm install
npm run dev
```

## Deploy on Vercel

1. Upload this folder to your GitHub repo.
2. Import the repo in Vercel.
3. Vercel should auto-detect Next.js.
4. Click Deploy.

## Add more keyboards

Edit `data/keyboards.ts`.
